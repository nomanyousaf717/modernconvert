Help & FAQ

Get answers to common questions about image conversion

Getting Started

How do I convert images?

Simply drag and drop your images onto the converter area, or click "browse files" to select them. Choose your output format (AVIF, WebP, JPEG XL, etc.), adjust quality if needed, and click "Convert". Your converted files will download automatically.

What image formats are supported?

Input formats: JPEG, PNG, WebP, AVIF, GIF, BMP, TIFF

Output formats: JPEG XL, AVIF, WebP, JPEG, PNG

Is there a file size limit?

No! Since processing happens in your browser, there are no server-imposed file size limits. However, very large files (100MB+) may take longer to process and use more memory.

Format Information

What is JPEG XL?

JPEG XL is the newest image format offering 55-60% better compression than JPEG while maintaining the same quality. It supports both lossy and lossless compression, wide color gamuts, and progressive decoding.

What is AVIF?

AVIF (AV1 Image File Format) offers excellent compression, typically 50% smaller than JPEG with better quality. It's supported by most modern browsers and ideal for web use.

Which format should I choose?

  • JPEG XL: Best compression and future-proof, but limited browser support
  • AVIF: Great compression with good browser support
  • WebP: Solid compression with excellent browser support
  • JPEG: Universal compatibility, larger file sizes
  • PNG: Lossless quality, supports transparency

Privacy & Security

Are my images uploaded to your servers?

No! All processing happens entirely in your browser using WebAssembly. Your images never leave your device, ensuring complete privacy.

Can I use this offline?

Yes! After the initial page load, the converter works completely offline. Perfect for when you don't want to share sensitive images online.

Do you track my usage?

We use anonymous analytics to understand how many people use the service, but we never track what images you convert or any personal data.

Troubleshooting

Conversion failed - what went wrong?

Common issues and solutions:

  • Corrupted file: Try a different image file
  • Unsupported format: Check our supported formats list
  • Browser compatibility: Use Chrome, Firefox, or Safari for best results
  • Memory issues: Try smaller files or refresh the page

JPEG XL not working on my device?

JPEG XL support varies by browser and device. It works best on desktop Chrome and Firefox. On unsupported browsers, we'll automatically suggest WebP as an alternative.

Conversion is slow

Processing speed depends on your device's performance and image size. For faster results, try reducing image quality or using a more powerful device.
